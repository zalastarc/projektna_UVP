# Projektna naloga UVP

V okviru moje projektne naloge bom analizirala podatke iz spletne strani https://zotks.si/priznanja/logika/.

To je stran, na kateri so vsako leto objavljeni rezultati tekmovanja iz logike za različne tekmovalne skupine.

Bolj natančno, analizirala bom rezultate sedmih tekmovalnih skupin za šolska leta 2020/21 - 2023/24. Vsak rezultat sestoji iz šest informacij - skupina, tekmovalec, mentor, šola, točke in mesto.

V datoteki podatki_s_spleta.py so izluščeni podatki, s pogonom datoteke shranjevanje_podatkov_s_spleta.py pa se pokličejo funkcije iz podatki_s_spleta.py ter ustvari datoteka.csv. Sama analiza je izvedena v datoteki analiza.ipynb.

## Navodila za uporabo programa

Pred uporabo programa si je potrebno namestiti knjižnice pandas, requests, matplotrib in seaborn. Uporabljali bomo tudi nekatere druge, že vgrajene knjižnice.
